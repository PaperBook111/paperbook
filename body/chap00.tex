\section{课题背景}
\esection{Research Background}
%
科技的变革给人类的生产生活方式带来巨大改变\ucite{almeida2020challenges}。物联网（Internet of Things, IoT）是继计算机和互联网之后的第三次信息技术革命\ucite{dash2019internet}。 物联网的出现是人类迈向“智能时代”的重要推力之一。物联网是物理世界与信息世界的融合，具有实时性和交互性的特点\ucite{huang2020intelligent}。 目前物联网已深入传统领域，促使传统行业信息化、智能化，其中包括电网，交通，物流，家居，零售、仓储等领域\ucite{zhang2022design}。 随着5G、云计算、人工智能等技术的日益成熟，物联网终于迎来了蓬勃发展期\ucite{rejeb2022big}。2015年物联网的接入设备为60亿台，据高德纳公司预测，到2025 年物联网接入设备达到270亿台\ucite{gul2017survey}。据市场动态报告显示，2016年物联网全球市场规模为1570亿美元\ucite{portilla2019extreme}，而网络解决方案供应商思科预测，未来全球物联网的产业规模可达8万亿美元\ucite{cisco}。
%
对于迅猛发展的物联网，世界各国都相应出台了一系列政策为本国物联网发展保驾护航。我国为抢占这一科技新的制高点，早在2006年，国务院发布的《国家中长期科学和技术发展规划纲要（2006-2020年）》中首次提出物联网相关政策，其后在《“十三五”规划纲要》中把物联网列为国家战略性新兴产业之一，推进物联网的跨界融合、集成创新和规模化发展。

射频识别（Radio Frequency IDentification, RFID）技术作为物联网的关键技术之一\ucite{gupta2020overview}，是自动识别技术的一种，是实现物联世界、信息高速采集的工具，被认为是21世纪最具发展潜力的信息技术之一\ucite{al2020intelligence}。2006年，我国15个部委联合发布《中国RFID技术政策白皮书》确定了RFID的发展战略，为RFID发展创造良好的成长环境。2021年，国务院发布《“十四五”规划纲要》中明确指出：扩大RFID应用需求，深入推进服务业数字化转型，培育智慧物流、新零售等新增长点\ucite{gwy145}。2020年，我国RFID市场规模突破1200亿元，且RFID市场规模继续保持高速增长趋势，预计2022 年将进一步增长至1601 亿元\ucite{RFIDmarketinchina}。 
%
RFID具有非接触、数据存储量大、识别速度快、保密性强的优势，且已经在智慧交通、智能制造、资产管理、新零售等领域广泛使用，不仅为个人提供生活便捷，企业生产节约劳动力，提高生产效率，而且实现了数据的自动采集，助力企业战略规划的准确性。例如：已惠及全国的电子不停车收费系统（ETC）2020年已发展用户数量超过两亿\ucite{zou2021method}；美国零售巨头沃尔玛在要求供应商采用RFID托盘完善供应链之后，将全面启用RFID技术打造智慧超市，因其行业的影响力，目前美国零售业“几乎普遍”采用RFID技术\ucite{woermaRFID}。此外，北京大兴，深圳等机场也都上线了RFID航空行李全流程跟踪系统，使乘客可以随时随地轻松掌握行李行踪。服装行业的迪卡侬、Nike、安踏等企业陆续使用RFID技术来实现货物管理与大数据的搜集应用。
%
且随着RFID技术的成熟以及标签价格的继续下降，RFID的应用将越来越广泛，将会给企业与个人带来更多便利，推进物联网世界“万物互联”的进程。

RFID是利用无线射频信号在传播过程中的电磁耦合原理来实现目标对象信息和数据的传递，进而达到对目标对象的自动识别。RFID技术的开端始于二十世纪四十年代，直接继承了雷达利用电磁波的反射对空战中敌我双方战机的识别的原理，且在1948年哈里$\cdot$斯托克曼发表的《利用反射功率的通讯》科学论文奠定了RFID的理论基础。典型的RFID系统通常由后台服务器、阅读器、天线和若干标签组成。后台服务器是RFID系统的逻辑中枢，存储标签ID以及标签关联信息，并对标签返回信息进行逻辑处理；阅读器是RFID系统最关键的组件，负责指令的编码和发送，以及接收标签信号的调制解码；天线负责能量的辐射以及标签返回信号的捕获；标签存储关联物品的唯一标识，具有有限的运算能力。RFID系统的基本工作原理：阅读器发射特定频率的射频信号，在射频信号覆盖范围内的标签把电磁信号转换为电信号，标签被激活并将自身存储信息编码通过内置天线发送出去，阅读器接收到标签返回的调制信号，经过信号的解调解码后传送到后台服务器，后台服务器进行逻辑运算识别标签信息，并针对标签作出相应的处理。

RFID技术是公认的21世纪最有望取代条形码的技术\ucite{shyla2021implementation}。第一，条形码识别速率慢，只能单例识别，而RFID识别速率快，自动识别，无需人工干预，且一次可批量识别多个标签；第二，条形码只能在可见光、近距离、无遮挡条件下识别，而RFID利用电磁传播通信，不需要光照，可远距离，穿透障碍物识别标签；第三，条形码存储数据量小，不能更改，不可反复使用，而RFID标签有较大的数据存储空间，存储数据可被反复读写，重复利用。

RFID强大的技术优势得到工业界与产业界的普遍认同，且高校以及相关研究单位并未放松对RFID技术的挖掘，越来越多的理论研究以及应用实践得以提出，不仅使RFID工作效率得到很大改善，而且其功能也日渐增多。随着制造工艺的提高以及RFID的技术进步，标签价格大幅下降，RFID的规模也在不断增长。伴随而来的是产业规模的增大，单靠一个阅读器和少量标签难以实现产业规模化需求。因此大规模RFID系统中通常包括多个阅读器和数以万计的标签数量。RFID的规模的增长，必然引发诸多问题。例如，阅读器参数使用不合理造成系统能耗增加、系统性能下降，甚至出现错误的执行结果；标签冲突加剧引发系统带宽阻塞严重、吞吐量下降，延时增加；阅读器与标签通信中标签的冗余验证加重了RFID低带宽信道信息传输负担。因此，在分析和总结现有研究工作的基础上，本文从以上三个方面详细说明它们成为制约大规模RFID系统信息高性能传输的原因：

首先，阅读器参数使用不合理造成系统能耗增加、系统性能下降，甚至出现错误的执行结果；在多阅读器的RFID应用场景中，如何促使阅读器的协同工作，并根据标签分布状况实时调整阅读器参数设置对降低系统能耗、提高系统执行性能具有重要意义。其次，标签冲突加剧引发系统带宽阻塞严重、吞吐量下降，延时增加；在实时性要求比较高的应用场景中如何降低冲突和实现冲突信息的再利用是缓解RFID通信阻塞问题的思考方向。再者，阅读器与标签通信中标签的冗余验证加重了RFID低带宽信道信息传输负担；阅读器对标签的认证只能在标签无冲突的情况下才能实现，而对冲突的判断需要发送比较长的验证信息，而验证信息过长且应用单一成为阻碍RFID系统性能提升的又一难题。因此，在分析和总结现有研究工作的基础上，本文根据以上三个问题，从多阅读器多类别RFID系统中丢失标签检测、单类别RFID系统中丢失标签识别和大规模RFID系统中未知标签快速识别三个实际RFID应用方向提升信息传输的性能：

第一，阅读器参数使用不合理；

多阅读器多类别RFID系统中丢失标签检测时间效率低。在RFID系统的丢失标签检测问题的研究中，大多相关协议忽略了阅读器冲突导致的标签认证失败给检测结果带来假阴性的影响（把存在的标签误认为丢失），且在丢失标签检测过程中，各个阅读器简单的设置为统一的参数（广播帧长等于标签数量），忽略了标签的空间分布状态（单个阅读器监控范围内的标签数量远小于标签的总数量），过大的广播帧长给丢失标签检测带来巨大的时间延迟；对于多类别标签的RFID系统，先前的研究工作仍然采用逐一标签类别检测的低效方式\ucite{shahzad2015expecting,tan2010efficient,liu2020fast}，在检测过程中能被使用的单一时隙在时间帧中比例最大为$36.8\%$\ucite{wang2021fast}，空时隙与冲突时隙被直接跳过或废弃，空时隙与冲突时隙在广播帧中的比重最小为$63.2\%$，空时隙的比重为$36.8\%$，过多的空时隙和不能被利用的冲突时隙直接导致了时隙利用率低、时间延迟增加，严重影响多类别RFID系统丢失标签检测的时间效率。缩短广播帧，减小空时隙的比重，合理利用冲突时隙来提高信道利用率是提高多阅读器多类别RFID系统丢失标签检测信息高性能传输和协议执行效率的有效途径。

第二，标签冲突加剧；单类别RFID系统丢失标签识别中冲突时隙不能被利用。在基于Aloha协议的丢失标签识别协议中，只有广播帧中的单一时隙能被有效利用\ucite{yu2018missing,liu2013completely,li2013efficient}，映射入单一时隙的标签能够被成功识别，而冲突时隙内的标签不能被识别，因为冲突时隙内阅读器收到的是多个标签返回的混合信号，混合信号不能被解码出属于单个标签的返回信号，因此，让响应在冲突时隙内的标签重新选择区分位置是冲突时隙能够被利用的关键。

第三，阅读器与标签通信中标签的冗余信息验证；在未知标签识别的研究中，传统Aloha协议难以辨别冲突时隙中是否有未知标签存在。 传统Aloha协议虽然能够高效的检测冲突时隙，但对冲突时隙的应用仍然是空白。在传统未知标签识别协议中，只有未知标签选择时间帧中预测的空时隙映射才能断定此标签为未知标签并能成功获取ID\ucite{chen2018using,su2020partitioning}，对映射入预测为单一时隙的未知标签，传统Aloha协议只能断定有未知标签存在却不能获取其ID，而对映射入预测为冲突时隙的未知标签，传统Aloha协议无法侦测到未知标签的存在。总而言之，传统Aloha协议无法从冲突时隙中准确检测到未知标签的存在并成功获取ID信息。在传统Aloha协议中能实现冲突检测功能的标签返回信息是RN16，RN16的16位随机数单纯的用以判断时隙状态造成了资源浪费，因此拓展RN16的功能使其不仅具有冲突检测功能，更能区分冲突标签是实现冲突时隙再利用的有效途径。



\section{主要研究内容}
\esection{Main research content}
本文重点研究大规模RFID系统中信息高性能传输的方法。面向不同的RFID系统需求，我们将研究分为三个方向：多阅读器多类别RFID系统中丢失标签检测，单类别RFID系统中基于冲突时隙展开的丢失标签识别，大规模RFID系统中未知标签识别。具体的研究内容及研究意义总结如下:

\textbf{1)多阅读器多类别RFID系统中丢失标签并行检测。}丢失标签检测是RFID研究的重点问题之一，且有很多相关的算法被提出，但他们只是简单的假设RFID系统中只包含单个阅读器和单个类别的标签。然而，由于标签关联物品的价值、品类、生产商不同，标签可能被分为多个类别。单个阅读器的通信范围有限（既使一个阅读器能连接多跟天线），大规模的RFID系统通常包括多个阅读器。对于包含多个阅读器、多个标签类别的大规模的RFID系统，他们则采用逐一类别，所有阅读器采用统一参数设置的简单检测方式，空时隙的比例没有下降，冲突时隙仍然被直接废弃，这样不仅没有提升时隙利用率，而且造成丢失标签更大的时间延迟。
%
本文提出了一种针对多阅读器多类别RFID系统的丢失标签的并行检测协议，通过设置类别ID区分标签类别，且利用较少的广播帧长实现多个标签类别丢失标签的并行检测，利用曼彻斯特编码特性解码冲突时隙中的融合向量为多个分属每个类别标签的时隙占用向量，进而检测每个标签的状态（存在或丢失）。在多阅读器多类别RFID系统中，充分考虑每个标签类别空间分布，优化每个阅读器的参数设置，避免阅读器之间的冲突造成的标签识别失败。实验表明多阅读器多类别RFID系统中，我们的协议比现有协议丢失标签检测协议速率快$1.5$倍。

\textbf{2)单类别RFID系统中基于冲突时隙展开的丢失标签识别。}完全的丢失标签识别是RFID研究的经典问题之一。在仓储管理、供应链和物品追踪等应用领域中，运营商总是希望实时掌握物品状态（丢失或存在），如果物品有丢失，可根据丢失物品价值采取不同的应对手段。因此，完整、快速的丢失标签识别才能准确、及时的掌握全部物品的状态和丢失物品价值。通过研究现有有关丢失标签识别的研究成果，我们发现现有协议执行中空时隙所占用的比重过大，冲突时隙不能直接被转化利用，唯一能被利用的单一时隙占比只有$36.8\%$\ucite{kang2021ultra,liu2016top}。如Liu\ucite{liu2014multiple}、Shanzad\ucite{shahzad2016fast}等人利用多次哈希的方式来提升单一时隙的比例，每次哈希只留下单一时隙，空时隙与冲突时隙参与下一次的哈希运算，直至空时隙与冲突时隙变为单一时隙，虽然这种方式使单一时隙的比例增加，但需要发送额外的指引向量引导标签哈希种子的选择，且在单次哈希运算中单一时隙的比例并没有明显提高，空时隙的比例没有减少，冲突时隙仍然不能被转化利用。
%
本文提出了冲突时隙展开的标签识别算法，该算法不是将冲突时隙直接废弃，而是尽力转化为有用时隙。为了实现这一目标，该算法利用两层哈希运算。第一层，每个标签随机选择自己的响应时隙；第二层，冲突时隙内的每个标签响应一个相同长度的二进制字符串，且随机选择字符串中的一个位置设为`1'，其他位置全为`0'。如果预测的单一时隙以及冲突时隙中的响应二进制字符串中对应位置为`1'，而实际协议中收到的标签回复为`0'，则表明对应的标签丢失。理论与实践证明，该算法的时隙利用率更高，协议执行速率更快。

\textbf{3)大规模RFID系统中未知标签识别。}未知标签识别也是RFID研究的实用性问题之一。由于RFID市场规模的增大，RFID的应用已经深入到供应链系统、仓储管理和商品溯源追踪等诸多领域。但未知标签的存在可能会给此类系统带来不可预料的经济损失或安全隐患\ucite{gong2017efficient,liu2015sampling}，比如易挥发、有毒物品混入食品储藏区。因此，在RFID的研究中心快速的未知标签识别具有重要且实际的意义。观察以往基于传统的Aloha未知标签识别协议，利用单一时隙灭活已知标签，利用空时隙识别未知标签，未知标签的识别效率直接与空时隙的比例相关。为了增加空时隙的比例，CSD（Collision Seeking Detection）\ucite{liu2020fast}利用后台服务器的高性能计算能力获取完美哈希种子，让已知标签映射入少量时隙，从而使大量时隙变为空时隙，这样虽然增大了空时隙的比例，但随着协议的执行，已知标签的数量并未减少，已知标签的存在使后台服务器计算复杂度增高，消耗能量增加。SBF-UDP（Sampling Bloom Filter-based Unknown tag Detection Protocol）\ucite{liu2015sampling}等协议利用采样的方式允许少量已知标签参与未知标签识别，但也不能改变传统Aloha协议只能检测冲突，无法有效利用冲突的本质。
%
本文提出一种冲突分解法，改变传统Aloha协议中标签返回阅读器RN16的规则，把RN16分为区分码和随机码两部分，与传统RN16相比，改进的RN16不仅具有传统RN16冲突检测的功能，而且能够提供区分冲突标签的机会。冲突标签重新选择区分位，让映射入冲突时隙的未知标签有了第二次被发现的机会，因此，利用改进的RN16，不仅可以利用原有的空时隙识别未知标签，而且冲突时隙内的未知标签也有了被检测识别的可能，增大了一个时间帧中未知标签被识别的概率，提升了未知标签识别的效率。

通过对大规模RFID中信息传输的探索研究，我们希望在此三方面取得技术突破，提升RFID系统性能，为RFID的技术发展贡献力量，为中国《“十四五”规划》中物联网的更广阔应用推广继续努力。

\section{论文组织结构}
\esection{Structure of This Thesis}
本论文共分为六章，主要的内容及组织结构如下：
%
第一章为绪论部分，主要阐明了本文的研究背景、主要的研究内容以及主要创新点。
%
第二章简单介绍了RFID技术的基本原理和RFID的发展历史，并从RFID研究的三个方面阐述RFID信息传输现有工作的总结与讨论。
%
第三章介绍了如何在多阅读器多类别RFID系统中快速实现丢失标签检测。我们利用阅读器调度方案避免R2Rc影响，同时根据各类别标签的分布状态把繁重的检测任务划分为多个轻量级任务，实现阅读器参数个性化设置，降低了丢失丢失标签检测中的假阴性概率，提高了检测效率。
%
第四章介绍了在丢失标签识别过程中如何把冲突时隙转化为伪单一时隙。我们根据冲突时隙内标签数量展开冲突时隙，让冲突时隙内标签有了第二次被区分的可能，冲突时隙的被利用增大了时隙利用率和现存标签的灭活效率的双重目标。
%
第五章介绍了如何改进传统Aloha协议使其对未知标签识别效率更高。我们改进标签回复阅读器RN16的结构，使其不仅具备检测冲突的功能，同时具有区分冲突标签、识别未知标签的目的。
%
第六章总结来本文的主要研究内容，并对未来的研究工作进行了展望。
